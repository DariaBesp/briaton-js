# Briaton JS - Каталог интернет-магазина светильников

## 🌐 Демо

https://dariabesp.github.io/briaton-js/

## 🚀 Запуск проекта

1. Клонировать репозиторий:
git clone https://github.com/DariaBesp/briaton-js.git
cd briaton-js

2. Запустить локальный сервер:
Если установлен VS Code, нажмите правой кнопкой на index.html → Open with Live Server

или
npx serve

## 🛠️ Технологии

- HTML5
- SCSS (Sass)
- JavaScript (Vanilla JS)
- JSON (для хранения данных)
- [Swiper.js](https://swiperjs.com/) - слайдер товаров
- [Just Validate](https://just-validate.dev/) - валидация форм
- [Tippy.js](https://popper.js.org/tippy.js/) - всплывающие подсказки

## ✨ Основные возможности

### Управление товарами
- **Динамическая отрисовка** товаров из JSON формата
- **Фильтрация** по типам и наличию товаров
- **Сортировка** по цене и популярности
- **Пагинация** для удобного просмотра каталога

### Корзина покупок
- Добавление и удаление товаров
- Автоматический подсчёт суммы и количества товаров
- Сохранение состояния корзины (localStorage)

### Пользовательский интерфейс
- **Слайдер товаров** с использованием Swiper.js
- **FAQ аккордеон** для часто задаваемых вопросов
- **Всплывающие подсказки** (Tippy.js)

### Валидация
- **Валидация форм** с использованием Just Validate

## 📁 Структура проекта

```
briaton-js/
├── index.html          # Основной HTML файл
├── css/                # Скомпилированные CSS файлы
├── scss/               # Исходные SCSS файлы
├── js/                 # JavaScript файлы
├── data/               # JSON данные товаров
├── images/             # Изображения товаров
└── fonts/              # Шрифты проекта
```

## 📌 Особенности реализации
- Данные загружаются из JSON (имитация API)
- Состояние корзины хранится в localStorage
- Модульная структура JS
- SCSS разбит на компоненты
